2.
A using declaration makes available a single name from a namespace,and it has the
scope corresponding to the declarative region in which the using declaration occurs. 
A using directive makes available all the names in a namespace.When you use a 
using directive,it is as if you have declared the names in the smallest declarative 
region containing both the using declaration and the namespace itself.
